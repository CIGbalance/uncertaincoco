Detailed Description

template<typename T>
class COCOfunc< T >

Wrapper for functions in COCO.

This class provides a convenient interface to a function in the COCO framework. It allows access to various of the original functionalities and adds features such as logging and cheater evaluations.

Cheater evaluations are evaluations of the COCO function that are not counted through the COCO framework. They are never accessible to the optimisation algorithms and are solely logged along with predicted values in order to be able to compute errors.
